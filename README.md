# Go Backend

Build backend pure Go. this is repository example, how to build backend service without any framework

# Requirements

  - Golang version 1.7+
  - Glide; click https://github.com/Masterminds/glide

# Installing

  - Install Glide https://github.com/Masterminds/glide
  - Install Dependecies
    ```shell
    $ glide install
    ```
# Running Test

  - Unit test
    ```shell
    $ make test
    ```

# Format Code
  ```shell
  $ make format
  ```
